Several companion modules extend Layout Builder's site-builder surface. Layout Builder Restrictions controls which blocks are available per bundle or region.

Layout Builder Styles adds UI-managed CSS classes to sections and blocks. Layout Builder Component Attributes allows HTML attributes to be configured on components.
